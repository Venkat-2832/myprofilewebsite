# Personal Portfolio Website

This is my personal portfolio website built to showcase my profile, skills, and projects.

## Tech Stack
- Frontend: React.js, Material UI
- Backend: Node.js, Express.js
- Database: PostgreSQL
- State Management: React Context API
- Styling: CSS, MUI
- Tools: Axios, Postman, DBeaver

## Features Implemented
- Responsive header with mobile hamburger menu
- Dark / Light theme toggle using Context API
- Mouse trail (cursor tracking) animation using Canvas
- Skills section connected to backend API
- Clean and modular project structure
