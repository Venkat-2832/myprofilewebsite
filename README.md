# Frontend – Portfolio Website

This is the frontend of my personal portfolio website built using React.js.

## Tech Stack
- React.js
- Material UI (MUI)
- JavaScript
- Axios
- CSS
- Context API

## Features
- Responsive navigation bar (desktop & mobile)
- Dark and Light theme toggle using Context API
- Hamburger menu for mobile view
- Interactive mouse trail effect using HTML Canvas
- Clean UI using Material UI components
